# 1주차 개발일지   

중점 사항
------------ 
1. 문자열에는 ''를 붙여야 한다. 다른 데이터형에는 생략
2. like -> 문자열 패턴
3. between ~ and ~ -> 사이의 데이터 출력
4. 인덱스 명 in (x,y,z) -> x,y,z 의 데이터 출력

과제 결과
------------
select * from orders where course_title = '앱개발 종합반' and payment_method = 'kakaopay' and email like '%naver.com'
